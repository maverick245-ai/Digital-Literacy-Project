📌 Project Overview

This project is part of the CSE0001 – Digital Literacy course at VIT Bhopal University.

As a Student Digital Ambassador, the objective of this project is to build awareness about digital literacy, create a professional digital presence, and promote safe and responsible use of online platforms.

The project is divided into five modules, each focusing on a specific aspect of digital skills.
